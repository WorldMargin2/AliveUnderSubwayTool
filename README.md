# AliveUnderSubwayTool

地铁生存游戏手续费计算器，帮助玩家计算药品和装备交易的手续费，优化交易策略。

## 功能介绍

本工具提供以下功能：

1. **药品手续费计算器**：根据药品价格计算交易手续费
2. **装备手续费计算器**：根据装备价值、词条数和统一回收价计算交易手续费
3. **可视化图表**：展示手续费变化趋势，帮助玩家找到最优交易价格点
4. **实时计算**：输入价格后立即显示计算结果

## 使用方法

1. 打开`calculator.html`文件
2. 选择需要计算的类型（药品或装备）
3. 输入相关参数：
   - 对于药品：输入药品价格
   - 对于装备：输入期望售价、统一回收价和词条数
4. 查看计算结果和图表分析

## 手续费规则

### 药品手续费

| 价格（0~100万） | 手续费率 |
| --------------- | -------- |
| ≥0              | 4%       |
| ≥20万           | 6%       |
| ≥30万           | 8%       |
| ≥40万           | 12%      |
| ≥50万           | 15%      |
| ≥60万           | 23%      |
| ≥70万           | 30%      |
| ≥80万           | 40%      |
| ≥90万           | 50%      |

### 装备手续费

装备手续费根据统一收购价倍率和词条数计算：

| 1词条   | 2词条   | 3词条   | 4词条   | 手续费率 |
| ------- | ------- | ------- | ------- | -------- |
| 0-5     | 0-10    | 0~20    | 0~40    | 4%       |
| 5-7.5   | 10-15   | 20~40   | 40~80   | 6%       |
| 7.5-10  | 15-20   |         |         | 8%       |
| 10-12.5 | 20-25   | 40~80   | 80~95   | 12%      |
| 12.5-15 | 25-30   |         | 95~120  | 15%      |
| 15-17.5 | 30-35   |         | 120~140 | 23%      |
| 17.5-20 | 35-40   |         | 140~160 | 30%      |
| 20-22.5 | 40-45   | 80~90   | 160~180 | 40%      |
| 22.5-25 | 45-50   | 90~100  | 180+    | 50%      |
| 25+     | 50+     | 100+    |         | 80%      |
